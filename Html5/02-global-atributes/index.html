Global Atributes HTML5

Acesskey

O atributo acesskey gera uma tecla de atalho para um elemento no site
Exemplo: -> alt+key

Autocapitalize

autocapitalize serve para strings, definindo letras maiusculas e minusculas de inputs
digitados pelo usuario.
Exemplo:

O elemento deve receber um dos seguintes valores:

  • off ou none: Nenhuma capitalização é aplicada (por padrão as letras serão minúsculas)
  • on ou sentences: A primeira letra de cada sentença será por padrão uma letra maíuscula; todas as demais letras serão por padrão minúsculas.
  • words: A letra inicial de cada palavra serão por padrão maiúsculas; todas as demais letras serão por padrão minúsculas.
  • characters: Todas as letras deverão ser maiúsculas por padrão.

Spellcheck

spellcheck define se o elemento pode ser verificado por erros de ortografia
por ser booleano ele recebe apenas os valores true ou false. Exemplo: Não corrija esse textto

Translate

O atributo translate indica ao navegador se o elemento deve ser traduzido ou não (muito usado pelo google)
mas não suportado pelo Internet Explorer e Fire fox
ele recebe os valores yes ou no.

Language

lang serve para definir em qual idioma um texto foi escrito.
